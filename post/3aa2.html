其实这个想法是在前两天折腾gclone转存Google Drive转存的时候，看到了“谷歌网盘管家”这个BOT，当时就感觉很方便，也想弄一个这样的，但是无奈技术有限，本人小白一个，太高深的东西也看不懂。于是在网上搜索了一圈，以及在群友@SunRainSnow推荐，发现了两个配置简单，效果也还可以的方法，特此记录一下。

Tips：一杯茶一包烟，node 项目装一天。。。

遇到npm报错问题，自行百度吧，因为我也没解决~~有大佬懂得可以评论一下😂😂😂

推荐使用yarn，具体安装方法请看

下面开始正式教程

我是在 Debian 系统上操作的，其他系统请自行切换执行的命令。

TeleShellBot：

项目地址：
https://github.com/marknote/TeleShellBot

示例如图：

1、安装 npm

sudo apt-get instatll npm

2、安装 TeleShellBot

git clone https://github.com/marknote/TeleShellBot.git
cd TeleShellBo
npm install

3、配置

首先，打开Telegram，找到BotFather创建一个机器人，并获取机器人的Token
具体方法请自行搜索，这里不再赘述
然后搜索@get_id_bot，找到get id这个机器人，点击start在聊天框口出现的内容中，记下your chat ID

在TeleShellBot目录下，打开 config.js 文件

nano config.js 此处可以换成你喜欢的编辑器

配置内容如下图所示：

4、运行

输入以下命令：

npm start

或者：

node index.js

如果没有报错，打开TG，找到机器人，在聊天框输入命令并发送，即可看到命令运行结果。

然后Ctrl + C停止，使用screen命令将其运行在后台：

screen -S TeleShellBot

npm start

然后按键盘上的Ctrl + A + D 即可.







shell-bot

项目地址：

https://github.com/botgram/shell-bot

该机器人还支持上传或下载文件，并且还具有一个简单的文本编辑器，方便使用。

示例如图：

1、安装

首先安装node-pty依赖项：

sudo apt install -y make python build-essential

安装shell-bot：

git clone https://github.com/botgram/shell-bot.git && cd shell-bot
npm install

2、启动

运行如下命令：

node server

第一次运行它时，它将询问您一些问题并自动创建配置文件：config.json。您也可以手动编写，请参见config.example.json。

启动后，它将在启动Bot ready.并运行时显示一条消息。为了方便起见，您可能需要与BotFather交谈并将命令列表设置为的内容commands.txt

3、后台运行

screen -S shell-bot

node server

然后按键盘上Ctrl + A + D
